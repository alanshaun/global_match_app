## 开发方案说明

### 架构设计
- **前端**：React 19 + TailwindCSS 4 + Wouter 路由
- **后端**：Express 4 + tRPC 11 + Drizzle ORM
- **数据库**：MySQL/TiDB
- **认证**：Manus OAuth
- **AI 集成**：内置 LLM API（产品分析、Cold Email 生成）
- **文件存储**：S3（产品图片、简历文件）

### 关键 API 集成
- **产品匹配**：使用 LLM 分析产品特征，调用搜索 API 获取公司信息
- **联系方式搜索**：优先从公司官网抓取，失败则从 LinkedIn 获取
- **职位搜索**：集成 Indeed、LinkedIn、Boss 等平台的搜索 API
- **简历解析**：使用 LLM 或 OCR 技术提取简历信息

### 数据流
1. **功能一流程**：产品上传 → AI 分析 → 公司匹配 → 联系方式搜索 → Cold Email 生成 → 结果展示
2. **功能二流程**：简历上传 → 简历解析 → 职位搜索 → 匹配度计算 → 时效过滤 → 结果排序展示

### 时间估算
- 数据库设计：1-2 小时
- 功能一后端：8-10 小时
- 功能二后端：8-10 小时
- 功能一前端：6-8 小时
- 功能二前端：6-8 小时
- 测试和优化：4-6 小时
- 总计：约 33-44 小时开发时间
